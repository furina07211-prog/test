# 项目整体分析报告（最终版，2026-03-27）

## 技术栈总结
- 后端：Spring Boot 3.2.5 + Spring Security + MyBatis-Plus 3.5.15 + MySQL 8.x + Knife4j/OpenAPI。
- 前端：Vue 3 + Vite 6 + Element Plus + Axios + Pinia + ECharts。
- AI：Python 脚本（`predict_cli.py`）+ Java 进程调用，支持 `prophet`（完整版）和 `simple`（简化加权移动平均）。

## 模块结构说明
- 后端分层完整：`entity / mapper / service / serviceImpl / controller / dto`。
- 业务模块：`auth`、`basic`、`inventory`、`purchase`、`sales`、`dashboard`、`ai`。
- 前端结构：`views` 页面 + `api` 请求层 + `stores` 登录状态 + `router` 路由守卫 + `components`。
- AI 链路：
  - `/api/ai/chat` 普通模式
  - `/api/ai/chat/stream` 流式模式
  - `/api/dashboard/forecast/run` + `/api/dashboard/inventory/optimize/run`
