# 基于SpringBoot+Vue的水果批发仓库管理系统论文章节框架

## 论文题目建议
基于 Spring Boot + Vue 的水果批发仓库管理系统设计与实现

## 摘要
- 研究背景：传统水果批发仓库管理普遍存在人工登记效率低、库存信息不透明、保质期管理粗放、低库存预警滞后等问题。
- 研究目标：设计并实现一个面向水果批发场景的仓库管理系统，完成采购入库、销售出库、库存盘点、库存预警等核心业务闭环。
- 技术路线：采用 Spring Boot 3、MyBatis-Plus、MySQL、Vue 3、Element Plus、ECharts 构建前后端分离系统，并引入大模型接口和 Python 销量预测算法实现 AI 增强。
- 主要成果：完成角色权限控制、基础资料管理、库存批次管理、采购销售业务、预警中心、智能助手、销量预测与采购建议、数据统计看板等功能。
- 结论：系统能够较好满足水果批发企业在仓储管理、业务协同和智能决策方面的需求，具有一定的实用价值与扩展意义。

## 第一章 绪论

### 1.1 研究背景
- 说明水果批发行业具有保鲜周期短、品类多、批次复杂、采购销售波动大的特点。
- 说明传统人工或半信息化管理方式在库存准确率、数据实时性、损耗控制方面存在不足。
- 引出信息化管理系统和智能预测技术在该场景下的必要性。

### 1.2 研究意义
- 理论意义：将仓储管理系统设计方法与 AI 预测、智能助手结合，扩展传统管理系统研究内容。
- 实践意义：提升库存透明度、降低损耗、增强采购决策科学性，提高水果批发企业运营效率。
- 教学意义：体现前后端分离架构、权限控制、数据库设计、算法集成、可视化分析等综合能力。

### 1.3 国内外研究现状
- 传统仓储管理系统研究现状：围绕库存台账、入库出库、基础资料、权限控制等展开。
- 智能库存预测研究现状：围绕时间序列预测、销量分析、安全库存模型、采购优化等展开。
- 大模型在业务系统中的应用现状：围绕自然语言问答、业务查询辅助、报表解释等展开。
- 归纳不足：传统系统智能化程度不足，很多系统缺少面向生鲜场景的批次保质期管理和 AI 辅助决策能力。

### 1.4 研究内容与论文结构
- 研究内容：
  - 构建水果批发仓库管理系统总体架构。
  - 设计用户权限、基础资料、采购、销售、库存、预警等业务模块。
  - 集成 AI 智能助手、销量预测与采购建议功能。
  - 对系统进行测试验证并分析结果。
- 简述各章节安排。

- 经济可行性：采用开源框架和本地部署方案，开发成本和运行成本较低。

### 3.4 系统安全设计
- 登录认证：基于 JWT 的身份认证。
- 权限控制：基于角色的接口访问控制。
- 跨域处理：统一 CORS 配置。
- 异常处理：统一返回体与全局异常拦截。

### 3.5 AI 功能设计

#### 3.5.1 智能助手设计
- 采用 OpenAI 兼容接口对接大模型。
- 后端封装 `AiService`，支持流式和非流式对话。
- 通过意图识别与工具查询服务限制模型能力边界，避免自由 SQL。

#### 3.5.2 销量预测设计
- 输入：历史销售数据。
- 处理：优先使用 Prophet 进行时间序列预测，失败时回退加权移动平均算法。
- 输出：未来 7 天预测值、上下置信区间。

#### 3.5.3 库存优化设计
- 安全库存公式：最近 7 天日均销量 × 安全天数。
- 推荐采购量公式：预测需求 + 安全库存 - 当前库存 - 在途库存。
- 输出结果落库供前端看板展示。
