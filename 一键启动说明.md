# 一键启动说明

本说明适用于本地开发环境，按步骤执行即可直接运行，无需再手动改配置。

## 一、固定参数
- MySQL：`localhost:3306`
- 数据库：`fruit_warehouse_manage`
- 数据库账号：`root / 123456`
- 后端端口：`8080`
- 前端端口：`5173`

## 二、一键导入数据库（Windows / PowerShell）
在项目根目录执行：

```powershell
mysql -h 127.0.0.1 -P 3306 -u root -p123456 -e "CREATE DATABASE IF NOT EXISTS fruit_warehouse_manage DEFAULT CHARSET utf8mb4;"
mysql -h 127.0.0.1 -P 3306 -u root -p123456 fruit_warehouse_manage < sql/_sanitized_schema.sql
mysql -h 127.0.0.1 -P 3306 -u root -p123456 fruit_warehouse_manage < sql/_seed_ascii.sql
```

## 三、启动后端
```powershell
cd backend
mvn -q -DskipTests compile
mvn spring-boot:run
```

后端启动成功后，访问：
- `http://localhost:8080/doc.html`

## 四、启动前端
新开一个终端窗口执行：

```powershell
cd frontend
npm install --cache .npm-cache
npm run dev
```

前端地址：`http://localhost:5173`

## 五、登录账号
- 管理员：`admin / 123456`
- 仓库管理员：`warehouse / 123456`
- 销售员：`sales / 123456`

## 六、故障排查
1. `8080` 或 `5173` 被占用：关闭占用进程后重启。
2. 前端请求失败：确认后端已启动，且 `frontend/vite.config.js` 代理目标为 `http://localhost:8080`。
3. 数据库连接失败：确认 MySQL 服务已启动，root 密码是 `123456`。
4. AI 相关能力不可用：未配置模型 Key 仅影响 AI 增强功能，不影响系统主流程。
